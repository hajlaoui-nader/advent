# ✨ Advent of Code 2019 ✨

## My solutions to [Advent of Code 2019](https://adventofcode.com/2019/) written in [Haskell](https://www.haskell.org/)

### Running puzzles

To run all puzzles:

```
stack run
```

#### Running a single puzzle:

```
./run solve 01
```

Benchmark the puzzle:

```
./run bench 01
```

Test the puzzle:

```
./run test 01
```
